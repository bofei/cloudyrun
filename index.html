CloudyRun 云测试驱动平台。

  • 文档

CloudyRun 是一个跨平台跨浏览器的测试驱动工具， 基于 NodeJS、MongoDB 和 SocketIO 实现， 可用来高效地执行功能测试、单元测试和 JavaScript 命令测试。

以下是他的界面：

下面是一段典型的测试代码：

 
<script src="/cloudyrun-jasmine-pkg.js"></script>
<script>
    var result = !+'\v1' ? '不好意思，IE 用户。您拿多少火车票我们都不会让您上飞机的！'
                         : '看，灰机！！';
    CloudyRun.sendResult(result);
</script>

对应的 Demo 已经为您准备好了，点开右上角的 “云图标”，即刻在 CloudyRun 里执行吧！ 不出所料的话，执行结果是这样的吗？

此外，CloudyRun 还对我们熟悉的 Jasmine 进行了特殊的支持，典型的书写格式如下：

 
<script src="/cloudyrun-jasmine-pkg.js"></script>
<script>
    CloudyRun.runJasmine(function() {
        describe('jasmine test', function() {
            it('should be passed expect IE8', function() {
                expect(1).toEqual(!+'\v1' && 'prototype' in Image ? 0 : 1);
            });
        });
    });
</script>

对应的 Demo 在这里，同上，点开右上角的云图标，不出所料的话，执行结果是这样的吧。

浏览器在 CloudyRun 的世界里，只有想不到，没有测不到。他理论上支持所有平台的所有浏览器。 使用 CloudyRun, 能让你更关注于测试代码的编写，而无需费心如何让他跑起来。
