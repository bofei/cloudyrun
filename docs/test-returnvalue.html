返回值测试

返回值测试顾名思义，就是将你指定的返回值在控制台里显示出来。

首先要记住这个接口，这是 CloudyRun 用来往服务器端回传数据的：

CloudyRun.sendResult(YourMessage);

下面是一段典型的测试代码：

<script src="/cloudyrun-jasmine-pkg.js"></script>
<script>
    var result = !+'\v1' ? '不好意思，IE 用户。您拿多少火车票我们都不会让您上飞机的！'
                         : '看，灰机！！';
    CloudyRun.sendResult(result);
</script>

对应的 Demo 已经为您准备好了，点开右上角的 “云图标”，即刻在 CloudyRun 里执行吧！不出所料的话，执行结果是这样的吗？
