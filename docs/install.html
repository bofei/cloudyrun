安装

  1. 安装 mongoDB
  2. 安装 nodeJS 和 npm
  3. 用 npm 安装依赖包
    npm install socket.io express jade mongoose emailjs node-uuid underscore
npm install forever -g
  4. clone cloudyrun 到本地
    git clone git://github.com/sorrycc/cloudyrun.git
  5. 将 src/config.sample.js 改名为 src/config.js，并修改配置
  6. 用 forever 运行 cloudyrun
    forever start path/to/cloudyrun/src/app.js
